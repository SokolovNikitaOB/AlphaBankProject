Инструкция по запуску приложения:
1) Откройте файл application.properties и измените currencyconversion.api_id и giphy.api_id на свои.
2) Запустите приложение.
3) Выполнить запрос. Запросы следует отправлять по адресу http://localhost:8080/currency-converter/from/{from}/to/{to},
   где {from} и {to} значения базовой валюты и валюты, чье изменение мы смотрим.
       *При получении бесплатной версии api_id валют доступен в качестве базовой валюты только USD.
        Если у вас есть платные версии api_id можете в переменную {from} ставить любую валюту (которая есть на сервисе, разумеется).*
4) Наблюдайте гиф-картинку и изменение курса валюты.
